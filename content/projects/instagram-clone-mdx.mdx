## Instagram Clone

- Designed and developed a **functional Instagram clone** with key features:
  - **User login/signup**
  - **Post creation and deletion**
  - **Commenting system**
  - **Secure JWT authentication** for API security.
- Technologies used: **React.js, JavaScript, CSS, Python, FastAPI, MySQL**.
