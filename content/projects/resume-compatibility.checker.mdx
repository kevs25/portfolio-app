## Resume Compatibility Checker

Built a full-stack resume analyzer using hybrid ML algorithms (keyword matching, TF-IDF, sentence transformers) and OpenRouter API, achieving 85% accuracy in resume-job compatibility scoring with detailed skill gap analysis and recommendations.

### Features

- Resume-job compatibility scoring
- Skill gap analysis
- Detailed recommendations
- Hybrid ML algorithms:
  - Keyword matching
  - TF-IDF analysis
  - Sentence transformers
- OpenRouter API integration

- **[Live Demo](https://resume-compatibility-checker.vercel.app/)**

### Technologies Used

- **Frontend:** React, TypeScript, TailwindCSS
- **Backend:** Python, FastAPI
- **AI/ML:** OpenRouter API, Cursor
- **Algorithms:** Keyword matching, TF-IDF, Sentence transformers

### Key Achievements

- Achieved 85% accuracy in resume-job compatibility scoring
- Implemented hybrid ML approach for better accuracy
- Real-time analysis and recommendations
